# Nginx Stream 안내

외부망 서버와 내부망 서버가 모두 있는 사용자는 Nginx Stream으로 TCP 포트를 전달할 수 있다.

```nginx
stream {
    server {
        listen 31001;
        proxy_pass 192.168.219.105:25565;
    }
}
```

## 주의사항

- 이 설정은 사용자 서버의 Nginx 설정을 직접 수정해야 한다.
- 원격 자동 수정 대상이 아니라 안내 가이드로 제공한다.
- 적용 전 포트 충돌과 방화벽 허용 여부를 확인해야 한다.
