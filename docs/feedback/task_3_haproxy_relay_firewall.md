# task_3 피드백: 중계 서버 방화벽 포트 개방

## 피드백

- HAProxy가 설치되는 중계 서버에도 외부에서 들어오는 TCP/UDP 포트 방화벽 개방 동작이 필요하다.
- HAProxy 설정만 추가되고 방화벽이 닫혀 있으면 라우트가 정상 적용되어도 실제 접속은 실패한다.

## 반영 기준

- HAProxy route 적용 시 중계 서버에서 `ufw` 또는 `firewalld`가 감지되면 외부 포트를 함께 개방한다.
- 허용 CIDR이 설정된 경우 CIDR 제한 규칙으로 개방한다.
- 서버 삭제 또는 HAProxy route 제거 시 같은 포트 규칙을 정리한다.
