# task_3 피드백: Agent 확인 버튼 제거 후 문구 정리

## 피드백

- 서버 등록 흐름에서 `Agent 확인` 버튼은 더 이상 존재하지 않는다.
- 등록 차단 문구가 `Agent 확인 후 서버를 등록할 수 있습니다`로 남아 있으면 현재 UX와 맞지 않는다.

## 반영 기준

- 서버 등록 조건은 `Agent 설치/업데이트 후 Agent API 접근 확인`으로 표현한다.
- 등록 버튼 자체가 Agent API를 호출해 확인하므로 별도 `Agent 확인` 버튼을 안내하지 않는다.
- 서버 생성 readiness에서는 `Agent 상태 확인` 대신 자동 확인 흐름을 설명한다.
